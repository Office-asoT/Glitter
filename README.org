* 使い方

#+begin_src javascript
<template>
  <glitter
    v-bind:images="[
      "/path/to/image/1",
      { src: "/path/to/image/2", caption: "キャプションも指定できます" },
      "/path/to/image/3"
    ]"
  >
    <!-- ここが表紙として表示されます -->
    <img src="/path/to/top/image" />
  </glitter>
</template>

<script>
import Glitter from 'glitter'
import 'glitter/dist/Glitter.css'

export default {
  components: {
    Glitter
  }
}
</script>
#+end_src

* 開発

[[https://cli.vuejs.org/][Vue Cli]] が必要です。

** 前準備

#+begin_src shell
npm install
#+end_src

** 開発用サーバの起動

#+begin_src shell
npm run serve
#+end_src

** ビルド

#+begin_src shell

#+end_src

** テスト
